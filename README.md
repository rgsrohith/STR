## ‚ú® Project Overview

**STR (Security Tracking Rugged Bag)** is a robust and intelligent bag developed to enhance personal security through **real-time tracking** and **unauthorized access detection**. This project leverages **IoT** principles and the **ESP32** microcontroller to provide proactive security alerts and location logging via the cloud.

---

## üöÄ Key Features

| Feature | Description |
| :--- | :--- |
| **Real-Time Tracking** | Integrated GPS module allows for continuous location monitoring via a dedicated mobile application. |
| **Theft Detection** | A magnetic reed switch triggers immediate alerts and notifications if the bag is opened without authorization. |
| **Cloud Logging** | Location, status updates, and theft logs are securely uploaded to the **Blynk cloud** platform. |
| **Core Hardware** | Built around the **ESP32** for Wi-Fi capability, with the magnetic reed switch and a rechargeable battery. |
